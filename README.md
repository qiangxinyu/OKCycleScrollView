# OKCycleScrollView
title:轮播图 
---

## 使用方法：


```objectivec
  #import "OKCycleScrollView.h"
  
  ...
  
  添加轮播图到指定视图上
  
  /**
     *  添加轮播图到指定视图上
     *
     *  @param view                         指定视图
     *  @param frame                        在视图上的Frame
     *  @param delegate                     设置代理
     *  @param images                       加载的图片
     *
     *  @return 轮播图实例对象
     */
  [OKCycleScrollView createOKCycleScrollViewInView:view Frame:frame delegate:self images:images];
  
```
